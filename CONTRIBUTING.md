# Contributing to DTPE-core

Thank you for your interest in contributing.

## Guiding Principles
- Preserve advisory-only behavior
- Maintain explicit human control
- Prefer clarity over cleverness
- Avoid implicit capability expansion

## Workflow
- Open an Issue before major changes
- Keep pull requests focused and minimal
- Update schemas and documentation alongside changes

## Conduct
Be respectful.
Assume good intent.
Keep discussions technical and professional.
